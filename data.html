We @ Developer Student Club NUS are launching our very first Data Analytics Workshop Series, kicking off with an Intro to Data Analytics with Python Workshop!

Through this series, we aim to equip anyone with zero analytics or programming knowledge with the skills required to tackle the essential components of any data analytics project.

Data Analytics with Python

Python is our language of choice for Data Analytics. In our first introductory workshop, we'll be getting everyone up to speed with the fundamentals of programming in Python, as well as to set up a proper data science environment on your machine. From there, you'll have your bases covered to explore the rest of the data science ecosystem.

Topics covered: a) Programming in Python b) Setting up a Data Science Environment w/ Anaconda and Jupyter c) Utilizing third-party libraries d) Git for Version Control

Machine Learning

After you have done a large amount of work manipulating, analysing and cleaning your data, it is time for your data to do the work for you. In this 4th of 5-workshop series on Data Analytics, we tackle the most popular Machine Learning techniques out there and use them to create powerful models that allows us to make predictions with all those data.
But creating your model is only the first step, how do you know whether your model worked? That is where evaluating and validation comes into play. Join us to learn more about validation and see your model improve as it gets “trained”.

Project Catalyst

Have the next big data science idea? Find like-minded individuals to join your team, or join a team with an exciting idea!
Join us, in the last of 5-workshop series on Data Analytics, as we join forces to construct a project outline, consisting of a target problem statement, list of actionable tasks, manpower delegation, and metric of project success.
Teams will be guided on scoping the project outline to be specific, targeted, technically feasible and be provided with suggestions on possibly more useful/ relevant tools for their proposed approach.
Teams can then get started on their next big idea!
